→ Строит бессрочную историю изменения цен
→ Отдаёт всё через REST API и веб-интерфейс

Конвейер обработки

Расширяемость

Ядро знает только контракты (BaseParser, MatchResult). Новый формат = регистрация парсера в реестре.

Портативность

Один код на SQLite (dev, zero-config) и PostgreSQL (prod, docker-compose).

Очередь

Фоновый воркер на пуле потоков — масштабируется до Celery/RQ без смены контракта.
